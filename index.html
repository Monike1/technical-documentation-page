Introduction

CSS (Cascading Style Sheets) is a language that gives styles to a website. You should have basic knowledge of HTML to learn CSS. Every element, class or id can be taken from an HTML document and styled as a selector in CSS (a selector doesn’t have angle brackets unlike an html element). To style an element, say what property you want to style and how it should look like – give it a value. The property can be a color (color means a font color), background-color, width, height or even html or body. The value can be blue, white, 10px, 20px, etc. Some code editors (e.g. Brackets) will suggest the value names after you start typing it.

CSS syntax

CSS document is built of CSS rules. The structure of a CSS rule looks like this: p {
color: red;
font-size: 16px;
}

Selector

When you look at the CSS rule above, there is a selector (e.g. p) and a declaration block (the curly braces and everything inside it). If you take any element from your HTML document and get rid of angle brackets, then you have a selector (this kind of selectors is called Type Selector). There are a few ways of selecting HTML elements in order to style them with CSS. See CSS selectors at MDN Web Docs. By learning basic selectors you will be able to style almost everything.

You can style more than one element in the same way by separating selectors with a comma. It would look like this: input, textarea, button, select, label {
font-size: 14px;
}

Declaration block
{color: red;
font-size: 16px;
}

/* you see above a declaration block */
/* make sure to place a selector before a declaration block */

Besides a selector there is also a declaration block – the curly braces and everything inside it. Each line inside a declaration block is called a declaration. Declaration is a property – value pair. Each property – value pair is separated by a colon. Each declaration is ended with a semicolon. CSS declaration looks like this:

color: red;

/* you see above a declaration, with a property of color and a value of red */
/* declarations wrapped in curly braces create a declaration block */

How do you know, what property to type in? There’s a list of most common CSS properties at MDN Web Docks.

Comments in CSS

The only official way of writing comments in CSS is by wrapping one or more lines of text or code in /* and */.

/* single line comment */

/* multiple lines
comment */
Adding CSS to a website

There are three ways of adding CSS to a website. Here is the most efficient and most used method, called external CSS. Basically you have a CSS document with .css file extension, e.g. style.css. It may be placed in the same directory as your index.html file. Now link your HTML document to your CSS styles. Place this link before the head closing tag of your HTML document. The path to your stylesheet may be different. It depends on directories where the CSS and HTML documents are placed. A link to your CSS document should look like this:

<link rel=”stylesheet” href=”style.css”>
